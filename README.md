# Ouidesign Premium Website

## Deploy to GitHub Pages

1. Create a GitHub repository and push this project.
2. In GitHub, open **Settings → Pages**.
3. Set **Source** to **GitHub Actions**.
4. Commit the workflow file and push to `main`.
5. The GitHub Actions workflow will build and publish the site.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
